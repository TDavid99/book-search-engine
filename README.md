# Book Search 

# Description 
For this application a user can sign up and login search for books. User can also delete books after they have read it. MERN stack app. 

#Technologies used 
Express.js
Mongoose
MongoDB
React
GraphQL
Bootstrap
Google Books API
Apollo
Node.js
Javascript Web Token
